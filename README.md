# Letter

Show that special someone that you are thinking about them with an interactive, handheld, battery-powered device. Images pulled from a micro SD card.

### Materials

* x1 Arduino Nano
* x1 ST7735 TFT LCD
* x3 Buttons
* x3 10k ohm resistors
* x1 9V battery
